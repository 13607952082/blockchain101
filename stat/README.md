# 统计小工具

## 统计学生issues的相关信息

功能：
* 统计面板
    * 学生创建的issues总数API
        * 有效的问题总数
        * 有效的评论总数
        * 参与的学生数
        * 已完成的学生数
* 搜索窗口
* 搜索结果窗口
    * 根据昵称或课程代号查询单个学生的issues信息API
        * 当前issues得分
        * 创建的问题总数
        * 创建的评论总数
        * 有效的问题总数
        * 有效的评论总数
        * 无效的问题总数
        * 无效的评论总数
        * 创建的问题列表
        * 创建的评论列表
        * 有效的问题列表
        * 有效的评论列表
        * 无效的问题列表
        * 无效的评论列表
